# FeN 双臂机器人

> **FeN**（铁氮）是 [Panbotica（全世萝卜）](http://www.panbotica.com) 研发的高精度双臂服务机器人，采用两条 UFactory xArm 机械臂集成，形成 12~14 自由度的双臂系统，广泛应用于精密服务场景。

## 核心特性

### 精密运动控制
搭载高精度伺服电机与先进运动控制算法，实现 **±0.1mm 重复定位精度**，可执行调酒、手冲咖啡、功夫茶等高精度操作。

### 强载荷能力
每臂最大有效载荷 **5kg**，双臂合计 10kg，可轻松应对重物搬运和高强度工况。

### 高效能源管理
内置高效能源管理系统，支持高峰时段长时间连续作业，降低维护频率。

### 环境自适应
配备高适应性传感器与视觉系统，在不同光线与环境条件下均可稳定工作。

### 安全可靠
内置急停按钮与故障检测系统，保障操作人员与客户安全。

## 机械臂配置（本项目）

FeN 由两条独立的 **UFactory xArm 6 / xArm 7** 机械臂组成：

| | 左臂 | 右臂 |
|--|------|------|
| 型号 | xArm 6 或 xArm 7 | xArm 6 或 xArm 7 |
| 自由度 | 6 或 7 | 6 或 7 |
| 控制器 IP（默认）| `192.168.1.202` | `192.168.1.201` |
| 通信协议 | TCP/IP（端口 18333）| TCP/IP（端口 18333）|

双臂通过同一局域网接入中控 PC（Ubuntu + ROS），由 `fen_bridge.py` 统一调度。
